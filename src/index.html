The VAARS

I successfully completed my final year project, Vehicle Accident Alert and Rescue Solution (VAARS), at COMSATS University Lahore. VAARS is a comprehensive system designed to enhance road safety and provide timely assistance in case of vehicle accidents. The project earned the first position across the university for its innovation and practical application
Introduction:
VAARS is designed to automatically detect vehicle accidents and promptly alert emergency services, reducing response time and potentially saving lives. The system integrates hardware and software components to provide a seamless and efficient rescue solution.
Key Features:
1. Accident Detection
2. Location Tracking
3. Communication.
4. Alcohol Detection System
5. Fog Detection System
6. Emergency Alert System
7. Mobile Application Developed a React Native application to display accident alerts and vehicle data, providing a user-friendly interface for users and emergency responders. Technologies Used:
- Software: React Native, Firebase, Node.js, and Google Maps API.
- Tools: Visual Studio Code, Git, Figma, and Firebase Realtime Database.
